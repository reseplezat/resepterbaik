Bahan-bahan dan bumbu yang perlu dipersiapkan untuk membuat Ayam Korea Pedas Keju Mozzarella (Ala OJJU) チーズタッカルビ:

  1. 600-700 g Paha ayam (potong ukuran bite size) // 鶏もも(一口大に切る) // Chicken Tight (Slice to bite size)
  2. 250-300 g Kubis // キャベツ(ザク切り) // Cabbage
  3. 100-150 g Ubi(Potong menjadi strip) // サツマイモ(短冊切り) // Yam
  4. 100-150 g Wortel(Potong menjadi strip) // ニンジン(サツマイモと同じ) // Carrot (Cut into strip)
  5. 100-150 g Bawang Bombay (Potong menjadi strip) // 玉ねぎ // Onion (Cut into strip size)
  6. 30 g Daun Bawang // ニラ// Leek
  7. 30 g Gochujang // コチュジャン ★★★
  8. 30 g Kecap // 醤油 // Soy sauce ★★★
  9. 10 g Bawang putih parut // ニンニクすりおろし// Grated Onion ★★★
  10. 5 g Jahe parut //しょうがすりおろし // Grated Ginger ★★★
  11. 5 g Bubuk Cabe // チリパウダー // Chilli powder ★★★
  12. 5 g Bubuk Paprika // パプリカパウダー // Paprika powder ●●●
  13. 30 g Sake // 酒 ●●●
  14. 15 g Gula // 砂糖 // Sugar ●●●
  15. 100 g Keju mozzarella // モッッアレラチーズ // Mozzarella cheese

Langkah-langkah untuk membuat Ayam Korea Pedas Keju Mozzarella (Ala OJJU) チーズタッカルビ

1
Campur ayam dan bumbu yang berlogo ★★★ 鶏肉と星を混ぜる。 Mix chicken and ★★★ logo ingredients
2
Taruh ayam di panci dan masak. Saat ayam sudah mulai berubah warna kecoklatan, tambahkan semua sayuran フライパンに鶏肉を入れて焼く。鶏肉の色が変わったら野菜をすべて入れる。 Put chicken in a pan and bake. When the chicken changes color, add all the vegetables
3
Taruh semua bahan berlogo bulat ●●● dan masak dengan api kecil selama 5 menit. Tutup dengan panci tutup ●●● をすべて入れる。蓋をして弱火で5分程度蒸し焼きにする。 Put all ●●● ingredients. Cover and cook on low heat for about 5 minutes
4
Saat ayam dan sayuran sudah matang, buat bulatan di tengah dan taruh keju 鶏肉と野菜に火が通ったらチーズを入れる隙間を作る。 When the chicken and vegetables are cooked, make a space for the cheese.
5
Saat keju sudah meleleh, artinya sudah bisa dinikmati! チーズが柔らかくなったら、出来上がり！ When the cheese is soft, it's done!
